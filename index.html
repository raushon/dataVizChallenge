Answer: - A horizontal bar chart is better than a scatter plot for this data because it is easier to compare the runs per game for each stadium. A scatter plot would be more difficult to compare because it would be harder to see the difference between the runs per game for each stadium. - I will choose a vertical bar chart over a horizontal bar chart when the number of categories is large and the labels are long. This is because a vertical bar chart will be more space efficient and easier to read.
